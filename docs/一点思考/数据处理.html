数据处理 ​

有些方法忘记了，在这记下

map ​

js
const newArr = arr.map(item => {
        return {
            role: item.id[2],
            content: item.kwargs.content
        }
    })

对这个数组使用，处理每一个项目并且返回到一个新的数组

filter过滤器 ​

js
this.session = this.session.filter(item=>{
        return item.sessionId !== sessionId
    }

过滤掉不需要的，留下需要的，这里写的就是留下来的条件了

match截取文本 ​

ts
let chatText:Array<string> = data.text.match(/“([^”]*)”/g);

这个是切分的，用正则表达式，这里截取“”里面的内容

  • “ – 匹配左中文双引号 “
  • ([^”]*) – 这是一个捕获组，匹配任意长度的、不包含右引号 ” 的字符序列
    • [^”]：表示除 ” 之外的任意字符
    • *：前一个模式出现 0 次或多次（尽可能多）
  • ” – 匹配右中文双引号 ”
  • /g – 全局匹配，找出字符串中所有符合的子串，而不是停在第一个 这个返回一个数组

join拼接 ​

ts
const audioText:string = chatText.join('。')

用。拼接

Object.assign() 复制对象 ​

浅拷贝的复制方式：

ts
const obj1 = Object.assign({},obj)

//其实就相当于：
const obj1 = obj

还有一种用法：

diff
+ const obj1 = Object.assign(obj)

//其实就相当于：
- const obj1 = obj

这个就是完完全全的对这个对象的引用了，地址都是一样的

深拷贝/浅拷贝 ​

就像上面的一样，对一个对象进行拷贝有很多方式，但是会被分出深拷贝/浅拷贝。这些都创建了新的对象

浅拷贝 ​

浅拷贝就是只拷贝了对象的第一层，碰到有嵌套的对象，它会直接把嵌套对象的地址值拿过来，因此你改变嵌套对象里面的属性，源对象也会收到影响！

tsx
function App() {
  const [obj,SetObj] = useState({
    name:123,
    age:123,
    chilren:{
      name:666,
      age:888,
      chilren:{
      name:666,
      age:888
    }
    }
  })
  console.log(obj);
  const obj1 = Object.assign({},obj)
  console.log(obj1);
  
  const change = ()=>{
    obj1.name = 666
    obj1.chilren.name = 999
    console.log(obj);
  }

  return (
    <>
    <div>{obj.name}</div>
    <div>{obj.chilren.name}</div>
      <button onClick={change}>change</button>
    
    </>
  )
}

你去点击，打印的obj里面的chilren是被修改的

深拷贝 ​

有个骚操作：

ts
const obj1 = JSON.parse(JSON.stringify(obj))

这样就是完全独立的新对象了，深拷贝说白了就是碰到嵌套对象的时候，就深入进去遍历重新复制给新的对象里面的新的对象属性
