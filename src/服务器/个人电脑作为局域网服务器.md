# 个人电脑作为服务器
这里使用到的主要是 linux、docker、nginx

## linux的安装
1. 在官方下载长期支持的版本之后，使用烧录工具将下载好的光盘文件烧录到u盘中
2. 在高级启动选项中选择重启
3. 插入u盘使用u盘安装ubantu
4. 安装ubuntu
5. 设置好之后，重启，选择ubuntu启动

## docker安装
docker是什么？
Docker 是一个开源的应用容器引擎，基于 Go 语言 并遵从 Apache2.0 协议开源。
Docker 可以让开发者打包他们的应用以及依赖包到一个轻量级、可移植的容器中，然后发布到任何流行的 Linux 机器上，也可以实现虚拟化。
容器是完全使用沙箱机制，相互之间不会有任何接口（类似 iPhone 的 app）,更重要的是容器性能开销极低。

### 安装
1. 更新apt工具和索引
```
sudo apt-get update
```
```
sudo apt-get install \ca-certificates \curl \gnupg \lsb-release
```
2. 添加docker官方gpg密钥
```
curl -fsSL https://mirrors.aliyun.com/docker-ce/linux/ubuntu/gpg | sudo gpg --dearmor -o /usr/share/keyrings/docker-archive-keyring.gpg
```
3. 添加docker软件源
```
echo "deb [arch=$(dpkg --print-architecture) signed-by=/usr/share/keyrings/docker-archive-keyring.gpg] https://mirrors.aliyun.com/docker-ce/linux/ubuntu $(lsb_release -cs) stable" | sudo tee /etc/apt/sources.list.d/docker.list > /dev/null
```
4. 更新索引包，安装docker
```
sudo systemctl daemon-reload
sudo systemctl restart docker
```
5. 验证：检查docker服务状态
```
sudo systemctl status docker
```

如果显示running就没问题


## 拉取nginx镜像
用docker取拉取nginx镜像，这里由于很多镜像源都失效了，所以使用了渡渡鸟镜像
```
sudo docker pull swr.cn-north-4.myhuaweicloud.com/ddn-k8s/registry.k8s.io/nginx-slim:0.21
```
后面的是从渡渡鸟镜像上复制下来的，根据镜像不同要更改

执行启动命令
```
sudo docker run -d \
  --name my-nginx \
  -p 80:80 \
  -v ~/my-website:/usr/share/nginx/html \
  swr.cn-north-4.myhuaweicloud.com/ddn-k8s/registry.k8s.io/nginx-slim:0.21
```

现在在本地访问localhost，就能看到网页

## 部署自己的网页
1. 将文件夹准备好，获取到完整路径。比如：`/home/kfjr/web-test`
2. 重新挂载目录
```
sudo docker stop my-nginx
sudo docker rm my-nginx
```
3. 启动新的容器,这里使用`/home/kfjr/web-test`来演示，实际服务时替换成自己的就行
```
sudo docker run -d \
  --name my-nginx \
  -p 80:80 \
  -v /home/kfjr/web-test:/usr/share/nginx/html \
  swr.cn-north-4.myhuaweicloud.com/ddn-k8s/registry.k8s.io/nginx-slim:0.21
```
这里会返回一串哈希值，然后你可以看到已经部署的网页了
