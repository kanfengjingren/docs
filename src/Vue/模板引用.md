# 模板引用与useTemplateRef()

vue里面也需要一些直接对dom的操作，应该说是当时自己用到了但是还不知道是个什么回事，现在看到了。

比如当时在这里要使用一个滚动的效果
```js
<ul ref="scrollBottom">
    <li :class="item.role === 'AIMessage' ? 'agent' : 'human'" v-for="item in main.chatHistory">
        <div class="role">{{ item.role === 'AIMessage' ? 'ai' : '用户' }}</div>
        :
        <div class="text">{{ item.content }}</div>
    </li>
</ul>

const scrollBottom = ref<HTMLUListElement>()
watch(()=>main.chatHistory.length, () => {
    scrollToBottom(scrollBottom)
}, {
    flush: "post"
})
```

## ref 
ref是一个特殊的 attribute，和 v-for 章节中提到的 key 类似。它允许我们在一个特定的 DOM 元素或子组件实例被挂载后，获得对它的直接引用。这可能很有用，比如说在组件挂载时将焦点设置到一个 input 元素上，或在一个元素上初始化一个第三方库

## 访问模板引用
vue3.5之前用的就是上面那一套，3.5之后新增了一个 **要在组合式 API 中获取引用，我们可以使用辅助函数 useTemplateRef()**

```ts
<template>
  <input type="text" ref="my-input">
</template>

<script setup lang="ts">
import { onMounted, ref, useTemplateRef } from 'vue';

const my_input = useTemplateRef('my-input') //useTemplateRef里面的参数名要和模板里面写的一致

onMounted(()=>{
  my_input.value?.focus()
})
</script>

<style scoped>
</style>
```
如果是老式写法，那就是你的变量名要和ref模板里面的相同
