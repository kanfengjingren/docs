# 上传和下载

## 上传文件
上传文件需要用到包：multer
```sh
npm install multer -S 
npm install @types/multer -D  //声明文件
```

1. 在module里面导入上传的这个包,然后进行配置
```ts
@Module({
  imports:[MulterModule.register({
    storage:diskStorage({  //有打包操作
      destination:join(__dirname,'../images'),  //路径
      filename:(_,file,callback)=>{   //文件名   下划线 _ 是占位符，表示不需要使用该参数，但为了保持函数签名正确而存在。
        const filename = `${new Date().getTime() + extname(file.originalname)}`  
        return callback(null,filename)
      }
    })
  })],
  controllers: [UploadController],
  providers: [UploadService],
})
```

2. 在controller里面定义接口
```ts
@Post('album')
  @UseInterceptors(FileInterceptor('file'))  //中间件，单个文件上传  设置字段名
  upload(@UploadedFile() file){ //多个文件呢？

    console.log('file',file);
    return '出发吧'
    
  }
```
- 这里设置路由和接口的时候需要用到中间件 `UseInterceptors` 。传单个文件使用`FileInterceptor`,多个文件使用`FilesInterceptor`
- **file** 是这个字段名，前端请求接口的时候也要用这个**file**
  
3. 启动该服务之后能看到dist下面有新的image文件夹，然后前端上传之后就能看到上传的图片了

## 访问静态资源
在main.ts里面使用app调用方法
```ts
app.useStaticAssets(join(__dirname,'images'),{
    prefix:'/kfjr'
  })
```
访问http://localhost:3000/kfjr/1774321486302.png
**prefix:'/kfjr'** 指的是加入了kfjr前缀，不使用的话，那就是*http://localhost:3000/1774321486302.png*

## 下载
下载有两种方式，一个是直接下载文件，还有一种是流式下载压缩包

```ts
@Get('export')
  download(@Res() res:Response){
    console.log();
    let url = join(__dirname,'../images/1774321486302.png');
    res.download(url)
  }

  @Get('stream')
  async down(@Res() res:Response){
    const url = join(__dirname,'../images/1774321486302.png');
    const tarStream = new zip.Stream()
    tarStream.addEntry(url)
    res.setHeader('Content-Type','application/octet-stream')
    res.setHeader('Content-Disposition',`attachment;filename=kfjr`)

    tarStream.pipe(res)
  }
```
